Accessing JavaScript Objects

When we talk about accessing JavaScript objects, we are referring to the process of retrieving specific values stored within an object.
Objects consist of key-value pairs, where each key represents a property,
and the corresponding value represents the data associated with that property.

You can access the values of object properties using dot notation (person.name)
or square bracket notation (person['name']). Properties can also be added, modified, or deleted dynamically during runtime.
